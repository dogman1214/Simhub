# Simhub
A custom-designed macropad for my sim-racing use, and for general purpose as well.

# Features:
* 3D - printed custom case
* 3 x 4 switch arrangement
* EC11 rotary encoder, with a switch built in
* USB-C connectivity
# CAD Model:

Everything is secured in place with 3 M3 screws and bolts, with the bolts serving a dual purpose as the base legs. The case is to be 3d-printed, along with the knob for the rotary encoder.
